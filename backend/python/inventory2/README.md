#  Product Management API

A Django REST API for managing products using MongoDB, built with a clean **Controller-Service–Repository architecture**.

---

##  Features

-  CRUD APIs for products
-  MongoDB integration using MongoEngine
-  Service Layer (business logic separation)
-  Repository Layer (database abstraction)
-  Input validation & duplicate checks
-  Swagger API documentation
-  Unit & API test cases
-  Pagination support

---

##  Tech Stack

- **Backend:** Django, Django REST Framework  
- **Database:** MongoDB (MongoEngine)  
- **API Docs:** drf-yasg (Swagger)  
- **Testing:** Django TestCase
